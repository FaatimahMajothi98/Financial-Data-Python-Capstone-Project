# Financial Data Analysis with Python - Capstone Project
### By Faatimah Majothi

#### Introduction:
This project was created for the [Analyze Financial Data with Python](https://www.codecademy.com/enrolled/paths/finance-python) course on [Codecademy](https://www.codecademy.com/learn), with the intention of showcasing the skills developed through the course.
Financial data used in this project was sourced from [StockAnalysis.com](https://stockanalysis.com/stocks/industry/utilities-renewable/) and imported via NASDAC codes using the `yfinance` python library.

#### Project Goals:
- Practice accessing and downloading financial data
- Calculate financial statistics to understand the individual performance of an asset, how groups of assets are related, and how to allocate investments across assets for optimal performance
- Create a presentation that showcases a portfolio of recommended environmental stocks
- Providing the weights at which they may invest in each stock depending on their risk preference

#### Contents:
The complete project is available in the Jupyter notebook 'FinanceCapstone.ipynb'. The final presentation is available in the 'Financial Data Capstone Presentation.pptx' PowerPoint file. Downloading the file may be necessary for viewing.

All further information can be found in the Jupyter notebook, and any feedback would be greatly appreciated.
